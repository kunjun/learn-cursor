# Cursor - 更快速地构建软件

### 订阅等级

Cursor 提供多种订阅等级以满足您的需求。

- **业余**
  - 14天 **专业试用**
  - 50次慢速 `premium` 模型使用
  - 200次 `cursor-small` 使用
  - 2000次完成使用
- **专业**
  - 每月500次快速 `premium` 模型使用
  - 无限次慢速 `premium` 模型使用
  - 无限次 `cursor-small` 使用
  - 无限次完成完成
  - 每月10次 `Claude Opus` 使用
- **商业**
  - 使用信息与 `专业` 等级相同
  - 额外福利，详见定价页面

### 高级模型

GPT-4、GPT-4o 和 Claude 3.5 Sonnet 都计为 `premium` 模型。

### 专业试用

所有新用户都可以获得为期14天的专业试用，可以访问所有专业功能。14天试用期结束后，未升级的用户将恢复到业余计划。

### 快速和慢速请求

默认情况下，Cursor 服务器会尝试为所有用户提供快速 `premium` 模型请求。然而，在高峰时段，
用完快速 `premium` 信用的用户将被转移到慢速池，这本质上是一个等待快速 `premium` 请求变得可用的用户队列。

这个队列是公平的，Cursor 将尽一切可能保持队列尽可能短。
然而，如果您需要更多的快速 `premium` 信用并且不想等待，您可以在设置页面上添加更多请求。

### 检查您的使用情况

您可以在 Cursor 设置页面上检查您的使用情况。您也可以在 Cursor 应用程序内，
在 `Cursor 设置` > `常规` > `账户` 下，按 “管理订阅” 为专业用户，或 “管理” 为商业用户。

Cursor 使用情况每月重置，基于您的订阅开始日期。

### 可选的使用量定价

您可以选择使用量定价，用于超出您计划中包含的内容的请求，通过访问您的设置页面。

使用量定价细节：

- 使用量定价按日历月计费（不一定与您的账单周期相同，大致会在每月的 2-3 日计费）
- 如果您立即取消请求或请求出错，我们不会计入使用量
- 您可以配置一个硬性限制，您每月的支付永远不会超过硬性限制（对于使用量定价）
- 目前，使用量定价仅适用于 Claude 3 Opus 和一些长文本聊天模型。
- 目前，使用量定价仅适用于 `premium` 模型。
